# CVProject

Éste es un repositorio de práctica y portfolio con React y Tailwind que se divide en dos proyectos:
- E-Comerce_CV
- TO-DO App

## Proyectos incluidos

### E-Comerce_Cv
Proyecto portfolio inspirado en una home de ecommerce de patines, con React, Tailwind y un backend en Express para consumir imágenes desde la API de Pexels.

### TO-DO App
Aplicación de tareas TO-DO list desarrollada con React y Tailwind CSS para practicar estado, filtros, edición de tareas y persistencia con localStorage.


## Tecnologías
React, Vite, Tailwind, Express, JavaScript.

## Objetivo
Aprendizaje aplicado, portfolio, práctica real.

## Estado
En desarrollo, iterativo.
