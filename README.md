# Proyecto práctico TO-DO con React y TailwindCss
Mi aplicación TO-DO puede (lo mas importante):
---PRIMERA FASE---
 - Añadir tareas
 - Eliminar tareas
 - Editar tareas
 - Completar tareas (CheckBox)
 - Tiene filtros para mostrar las tareas (All, Active, Completed)

---SEGUNDA FASE---
 - Contiene localStorage
